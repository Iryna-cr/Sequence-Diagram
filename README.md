# Sequence-Diagram Завдання 3

## Тема - Оформлення замовлення в онлайн-магазині

## Зміст:
На діаграмі показано взаємодію між об’єктами *User, WebApp, OrderService, PaymentService, EmailService.*
Відображено послідовність викликів під час вибору товару, створення замовлення, оплати та надсилання email-підтвердження.
Застосовано синхронні виклики та зворотні відповіді.

## Показує:
Логіку виконання операції оформлення замовлення та послідовність обміну повідомленнями між компонентами системи.
